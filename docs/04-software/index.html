    常用软件推荐 #

    不同操作系统（无论电脑还是手机）支持的软件格式不一样，有些软件可能只限于 Windows 系统，而没有 macOS 版本；有些软件可能只限于 iOS 系统，而没有 Android 手机版本。

    无论针对哪个系统，我们都按照如下原则推荐软件。从上到下，优先级递减：

    • 如果可以不安装软件，就尽量不要安装软件，而使用在线服务；

    • 不推荐任何来自中国公司的软件和服务；

    • 优先选择使用人数较多的开源软件和自由软件；

    • 如果实在没有开源软件和自由软件可选，则推荐国外大公司的免费软件；

    • 最后推荐国外大公司的收费软件。

    • 即时通讯软件
