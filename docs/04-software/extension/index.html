    浏览器扩展 #

    以上推荐的几个浏览器都支持安装扩展，加强浏览器某些方面的功能。

    扩展可以使浏览器变得更强大更好用，但扩展本身也是一种软件，它可能也存在漏洞和风险，而且安装太多扩展可能拖慢浏览器的启动和运行速度。

    因此建议只安装最必需的扩展，比如以下这些与安全隐私有关的：

    一、适用于 Firefox, LibreWolf 和 Mullvad 浏览器 #

    1. uBlock Origin #

    LibreWolf 和 Mullvad 默认已包含。

    https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/

    2. NoScript #

    Mullvad 默认已包含。

    这个扩展的默认行为是阻止所有不在白名单中的脚本。这可能使那些依赖于 JavaScript 的网页无法正常显示。需要在使用过程中慢慢调整它的设置。

    https://addons.mozilla.org/en-US/firefox/addon/noscript/

    3. Privacy Badger #

    https://addons.mozilla.org/en-US/firefox/addon/privacy-badger17/

    4. HTTPS Everywhere #

    https://www.eff.org/https-everywhere

    5. Decentraleyes #

    https://addons.mozilla.org/en-US/firefox/addon/decentraleyes/

    6. Cookie Autodelete #

    https://addons.mozilla.org/en-US/firefox/addon/cookie-autodelete

    7. Disable WebRTC #

    https://addons.mozilla.org/en-US/firefox/addon/happy-bonobo-disable-webrtc/

    二、适用于 Brave 浏览器 #

    Brave 浏览器的扩展需要从 Chrome Web Store 下载。以上介绍的几种扩展都可以找到对应版本。

    1. uBlock Origin #

    https://chromewebstore.google.com/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm

    2. NoScript #

    https://chromewebstore.google.com/detail/noscript/doojmbjmlfjjnbmnoijecmcbfeoakpjm

    3. Privacy Badger #

    https://chromewebstore.google.com/detail/privacy-badger/pkehgijcmpdhfbdbbnkijodmdjhbjlgp

    4. HTTPS Everywhere #

    https://chromewebstore.google.com/detail/https-everywhere/ikclbgejgcbdlhjmckecmdljlpbhmbmf

    5. Decentraleyes #

    https://chromewebstore.google.com/detail/decentraleyes/ldpochfccmkkmhdbclfhpagapcfdljkj

    6. Cookie Autodelete #

    https://chrome.google.com/webstore/detail/cookie-autodelete/fhcgjolkccmbidfldomjliifgaodjagh/

    7. WebRTC Control #

    https://chromewebstore.google.com/detail/webrtc-control/fjkmabmdepjfammlpliljpnbhleegehm
