    办公和线上协作 #

    一般情况下，在线的办公与协作平台已经可以满足需求。

    Cryptpad #

    https://cryptpad.org/

    开源、端对端加密的协同办公工具。提供文档、表格、幻灯片、Markdown 等多种格式支持。

    可以不注册、不登录就使用。如果想要注册，不用提供邮箱或手机号码，只需填写用户名和密码即可。

    【请注意】一旦忘记密码，就不能再登录。因为它无法帮你重置密码。

    可以自己架设 Cryptpad 服务，也可以使用现有的服务，这些网站都基于 Cryptpad 搭建：

    • https://cryptpad.fr/ （近期有些不太稳定）
    • https://cryptpad.malacher.fr/
    • https://cryptpad.disroot.org/
    • https://cryptpad.private.coffee/
    • https://pad.envs.net/

    Riseup Pad #

    https://pad.riseup.net/

    Riseup 旗下服务之一，可以匿名使用，还能为文档设置有效期：一天，三个月或者一年。

    美中不足的是，它支持的富文本格式有限，只能进行简单的文档编辑。

    Google docs（谷歌文档） #

    https://docs.google.com

    Google 服务之一，需要有 Gmail 账号才能使用。这意味着，你需要使用手机号码来完成注册验证，这一步可能会暴露你的身份。请一定不要使用中国手机号码来注册。

    如果你非常在意隐私和数据安全，那么请谨慎选择 Google 服务。

    LibreOffice #

    https://www.libreoffice.org/

    如果你一定要在电脑里安装办公软件，推荐使用开源的 LibreOffice.

    微软 Office 套件 #

    如果必须使用微软的办公套件，请将其作为最后选项。

    Nextcloud #

    https://nextcloud.com/

    可以自己搭建服务的开源软件，包含一系列办公套件。

    不要使用 WPS 和其它国产软件 #

    一定不要使用 WPS 之类的国产办公软件。

    至于石墨文档、腾讯文档等国产在线办公平台，也不建议使用。原因不言自明：它们全都处于监控之下，你在上面创作的内容可能会给你带来风险，或者会被它们删除、屏蔽、禁止分享。
