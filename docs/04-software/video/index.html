    视频通话及会议 #

    Signal #

    https://signal.org/

    它不仅是即时聊天软件，也可以作为音频通话、视频通话和视频会议软件，支持多人的小组视频会议。

    关于 Signal 的详细介绍，请点击这里阅读。

    Jitsi #

    https://jitsi.org/

    开源的视频会议软件平台，可以自行搭建，也可以使用 Jitsi 提供的现有服务。 https://meet.jit.si/

    无须单独安装软件，只需要在浏览器里输入会议链接即可加入。

    ZOOM #

    https://zoom.us/

    商业、非开源的视频会议软件，支持所有操作系统。

    这可能是最流行的视频会议软件，但它曾经传出阻断中国异议人士讨论会的情况，建议谨慎选择。

    ZOOM 免费版本仅限最多 40 分钟通话，且在中国使用很不稳定。

    如果要使用 ZOOM，请从它的官方网站下载。国产安卓手机的应用商店搜索不到最新版本的 ZOOM。

    Webex #

    https://www.webex.com/

    商业、非开源的视频会议软件，支持所有操作系统，Cisco 公司旗下产品。

    可以作为 ZOOM 的替代，但 Cisco 公司曾经被曝参与中国 GFW 建设，建议谨慎选择。

    Webex 免费版本仅限最多 40 分钟通话，目前在中国使用暂且稳定。
