    喝茶之前 #

    一、可能会被喝茶的迹象 #

    参加过有特定敏感人物出席的活动，而且参加同一活动的伙伴已经被喝茶。比如，他们可能带着安装了国内软件——比如微信的手机到活动现场，导致数据泄露，也可能是其他原因导致信息泄露。

    国内的家人被社区、警察等找问话。

    自己收到当地使馆或警察电话（比如收到“注意言行”的警告；但不包括诈骗电话）。

    如果警察打电话来找，一般危险程度较低。如果是敏感程度高、危险程度高的事，警察会直接上门找。

    二、谁来了？是学校、社区、警察还是国保？ #

    在现场被监控系统识别出身份信息，如果情形不太严重，会反馈到主管单位，比如学校、工作单位等。

    如果你是个人或者在私企工作，通常会找你个人或者你所居住的社区。

    现已实现全民监控的环境，非敏感的活动人士也可能会被喝茶。

    三、来干嘛？聊天、约谈、传唤还是拘留？ #

    为什么有时候反应比较慢？比如隔了一周才来找。

    海关检查：国安系统； 进入海关之后：国保系统。

    国安和国保系统不一样。
