    准备事项 #

    一、日常电子设备使用习惯 #

    国产软件跟国外安全社交软件（包括但不限于 Telegram、Signal 等）不要混装在同一个手机上。否则，微信和其它国产软件可能盗取你的文件、图片，悄悄读取你的其它资料，甚至未经同意就打开你的手机麦克风权限等。也就是说，日常使用两部手机，其中一部只安装国产软件，另外一部只安装国外安全社交软件。

    而且建议，安装国外安全社交软件的那部手机最好不要是国产手机，比如华为、荣耀、小米等。如果无法做到使用国外品牌的手机，至少要保证国产软件和国外软件严格隔离。

    关于隔离电子设备，详细请看前文内容。

    最好使用苹果设备——司法实践显示技术苹果设备较难破解。但是根据 2024 年 1 月的新闻，北京司法鉴定中心的合作机构已经可以追踪 AirDrop 文件投递者的真实身份。详细请看前文内容。

    不要用日常手机号收取验证码，不要使用能够关联到真实身份的邮箱注册新的化名身份。

    也就是说，当你需要注册一个化名身份（无论邮箱还是社交网站）的时候，请从第一步就做到与真实身份的分隔，不要让它们之间有任何联系。详细请看前文内容。

    Telegram、WhatsApp、Signal、Twitter、邮箱等，不要使用自己的真实信息和日常联系方式。

    回国后，安全期之后再开始使用 Signal、Gmail 等软件。

    二、使用“空白”设备或清理原有设备 #

    不携带原有手机、电脑、iPad 回国。要么与信任的、安全的朋友交换手机；要么买一部“空白”的二手手机回国。

    把敏感资料备份在海外，手机和电脑格式化并重装系统。不要冒数据被恢复的风险，即使恢复的可能性只有 1%。

    Telegram 设置隐藏号码、加密聊天、两步认证。

    见警察的时候尽量不带手机。如果必须带手机， 喝茶的时候只带装有国产 App 的手机。

    三、提前准备律师委托书和其他资料 #

    1. 律师委托书 #

    把自己直系亲属的联系方式给国内外信任的朋友。因为根据法律，只有直系亲属（父母、兄弟、配偶、子女）才有权利为你委托律师。请注意，没有登记结婚的情侣无法为对方委托律师。

    自己委托：准备好自己签署的空白律师委托书和身份证资料交给国内外信任的朋友。空白律师委托书的模板可以从网上下载。

    律师必须要有你的委托书和身份证资料才能要求会见。

    被抓的 48-72 小时是黄金救援时间。在看守所会被警察密集的讯问和威逼利诱，律师在这期间可以给你很多建议，包括什么文件可以不签，什么问题可以不回答等等。

    2. 向亲友同步日程与行踪 #

    跟你最好的亲友报备航班落地时间、具体机场和行踪等。说明如果你在某个时间没有出现，则很有可能已经被警察带走。

    3. 提前录制自述视频 #

    自己录制一段视频，交给国内外亲友保管，万一被抓，即可公布视频。这种自述视频非常重要，可以引发媒体关注，对国际声援非常有用。

    比如曹芷馨被捕之前所录视频 https://www.rfa.org/mandarin/yataibaodao/renquanfazhi/ec-01172023041804.html

    四、在海外参加活动注意事项 #

    网络公开场合发表言论尽量用化名（包括 Twitter、Telegram 等各种场合）。

    线下现场活动佩戴口罩，以防现场有人（比如海外警察站或大使馆的线人）监控。
