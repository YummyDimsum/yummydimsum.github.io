    分享之前，删除元数据 #

    一、什么是“元数据”泄漏 #

    “元数据”英文叫 metadata，意思是关于数据的数据。不同文件类型包含的元数据量不同，一张照片可能包括上百项元数据，比如这些：

    • 拍摄设备型号
    • 拍摄时间（会带有时区信息）
    • 作者姓名/缩写
    • 公司/组织名称
    • 最后修改时间
    • 最后修改设备型号
    • 最后修改软件
    • 拍摄位置

    当你把图片、照片和文档发给别人的时候，或者分享到社交媒体的时候，如果没有清除这些元数据，它们就很有可能会泄漏你的很多个人信息。无论你在网上是公开活动还是匿名活动，都应该养成好的习惯，把图片和文档的元数据清除之后再分享出去。

    二、怎样防止“元数据”泄漏 #

    有一些软件可以帮你清理元数据，不同操作系统里的软件不一样，但操作都比较简单。

    • ExifTool
