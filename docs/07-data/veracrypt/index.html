    使用 Veracrypt 加密文件和磁盘 #

    Veracrypt 是一款开源的加密软件，支持 Windows, macOS 和 Linux 等操作系统。

    使用 Veracrypt, 能够将单个文件、文件夹加密，还可以将整个系统乃至整个移动存储设备加密。

    这是目前最为广泛使用的一款文件和磁盘加密软件，网站只有英文内容，安装之后可以设置为中文界面。

    https://www.veracrypt.fr/en/Downloads.html

    这里有一份详细的使用指南（繁体中文）

    http://wiki.kmu.edu.tw/index.php/%E7%A1%AC%E7%A2%9F%E9%9A%A8%E8%BA%AB%E7%A2%9F%E5%8A%A0%E5%AF%86-VeraCrypt%E8%BB%9F%E9%AB%94%E6%95%99%E5%AD%B8

    【特别提醒】在正式开始操作之前，请一定认真阅读使用指南，并将你的所有重要数据备份。
