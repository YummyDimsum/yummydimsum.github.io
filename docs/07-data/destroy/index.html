    使用 BleachBit 销毁文件 #

    有些时候，数据安全还包括销毁文件。一些文件如果落入黑客或者警方手中，可能会给你和你的伙伴带来极大风险。因此，只有加密保存还不够。这时你需要将那些敏感的、但却不再需要的资料彻底销毁。

    一般人的做法是在电脑上面点击“删除”，并清空回收桶（无论使用哪种操作系统，都有这样的功能）。

    但是，仅仅这样做完全不够！警方手中的机器可以在很短时间之内恢复这些数据。你需要更专业更强大的工具来帮你彻底清理和销毁文件。

    在这里我们推荐 BleachBit, 这也是电子前沿基金会（EFF）推荐的软件。

    最详细的指南，还是应该看 BleachBit 的官方文档。 https://docs.bleachbit.org/

    虽然它是全英文的，但有丰富的配图和视频演示。
