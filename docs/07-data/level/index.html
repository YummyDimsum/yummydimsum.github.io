    给你的数据重要性分级 #

    尽管我们的电脑、手机和硬盘里存储着大量数据，但我们只要稍微思考一下就会发现，这些数据的重要性并不一样。

    有些可能是网络上随便都能下载到有图片、视频，而且内容也并不敏感；有些可能是重要的个人资料，比如身份证、护照、毕业证、保险合同等资料；有些可能是你的公开身份所做工作的资料；还有些可能是你的匿名身份正在参与的一些敏感的事情……

    总之，稍微想几秒钟，你就会发现，这些信息的重要性和危险性都不一样。身份证照片如果被黑客偷走，确实会带来很大麻烦。但匿名参与敏感事件的信息如果被警察掌握，可能不只是麻烦而是风险。

    因此，在我们正式进入数据安全的相关软件工具之前，请先根据自己的实际情况，认真严肃地给你手上的数据做一次重要性和危险性的分级。

    这里我们只谈那些最有可能给你带来危险的数据，也就是你所从事的敏感活动相关的数据，它们可能包括：

    • 与你的在线虚拟身份有关的所有文件；

    • 你所从事的公民行动的所有文件；

    • 你的浏览器使用记录和缓存；

    • 各类聊天软件的本地记录和缓存等。

    敏感和重要的文件，最好不要保存在电脑硬盘上面，而是改用非常小的 U 盘保存，而且要将整个 U 盘完全加密。关于加密，详见后文关于 Veracrypt 的内容。
