    运行 GNU/Linux 系统的 PC 机 #

    Linux 平台的病毒、木马数量远远少于 Windows, 但这绝不意味着 Linux 系统百分之百安全，因此你也要为 Linux 系统做好相应的安全防护。

    这个世界上存在几百种不同的 GNU/Linux 发行版，我们很难列出每一种的安全设置方案。但是，无论哪种都有一些基础的、通用的注意事项。请参考这个说明：

    https://securityinabox.org/zh/phones-and-computers/linux/

    此外，建议仔细阅读所用 Linux 发行版的官方文档、官方 Wiki, 那里一般会有针对特定发行版的详细说明。
