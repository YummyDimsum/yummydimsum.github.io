    运行 Windows 系统的 PC 机 #

    一、决定购买或使用之前 #

    运行 Windows 系统的 PC 机是最常见的个人电脑。电脑硬件由不同的厂商生产，但操作系统大都是 Windows。

    如果你可以选择，建议你不要购买和使用中国公司生产的 PC 机，包括华为、小米、联想、神舟、荣耀、清华同方等等。相对而言，我们更推荐使用国外品牌，比如惠普、戴尔、三星、华硕等。但是如果你已经在使用中国公司生产的电脑，也不用过于担心，我们可以通过许多安全设置尽量避免一些问题。

    如果是从正规渠道购买的 PC 机，大多数电脑已经预装了操作系统，大部分情况下就是最新版的 Windows 系统，这是由电脑生产厂商定制的正版系统。

    如果你可以选择，建议购买内存（RAM）至少 16GB 的 PC 机，这样可以使电脑与软件运行更加顺畅。如果你现在使用的 PC 机内存不足 16GB, 可以咨询专业人士，确认是否能够增加内存。PC 机比苹果电脑方便的是，你可以更换配件来给电脑硬件升级，而不需要换掉整台机器。

    在 Windows 系统任务栏的搜索框中键入“信息”，点击选择“系统信息”，即可查看详细的硬件信息，包括 RAM 信息。

    二、盗版系统问题 #

    许多中国用户使用盗版的 Windows 系统，这样做存在安全隐患。常见的盗版系统可分为两种：

    1. 安装系统所用的文件来自微软官方，这种称作“原版系统”。系统本身没有问题，但在安装使用的时候，通过非正规渠道（比如从淘宝购买，或者使用破解软件）获得的激活码来取得正版授权。这种盗版虽然不道德，但是只有比较小的风险，风险来自破解软件可能携带恶意程序。

    2. 安装系统所用的文件本身就有问题，常见的是一些电脑维修人员自己制作的 GHOST 文件，而不是来自微软官方的系统安装文件。这种 GHOST 文件可能是原版系统（参见上一条）的镜像，也可能是由维修人员加入其他软件和工具的修改版。事实上，对于不熟悉电脑的人来说，也很难弄清楚使用 GHOST 文件安装的系统里都会包含什么。通过这种方式安装的 Windows 系统，存在很大的安全风险。因此我们强烈建议大家不要用这种方式安装系统，也要拒绝别人使用这种方式给你安装系统。
