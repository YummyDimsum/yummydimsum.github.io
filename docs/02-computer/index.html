    电脑硬件及操作系统 #

    无论你想要加强信息安全还是网络安全，首先都要确保电子设备本身的安全。如果电子设备本身存在重大安全问题，那么其它各种安全策略的效果也都会大打折扣。因此，我们将电子设备安全放第一部分。在这一章，我们首先讨论电脑的硬件和操作系统安全。

    对于大部分用户而言，常见的两类电脑就是：

    • 运行 Windows 或 GNU/Linux 的 PC 机；

    • 运行 macOS 系统的苹果电脑。

    之所以要单独讨论电脑硬件，是因为它直接关系到你使用哪种类型的操作系统，进而关系到你使用哪些应用软件，当然也就关系到信息安全和网络安全。

    这一部分包含以下几个小节：

    • 运行 Windows 系统的 PC 机

    • 运行 macOS 的苹果电脑

    • 运行 GNU/Linux 系统的 PC 机

    • 使用 Live USB 系统

    然而，信息安全和网络安全是一个全面、系统的工作，做好操作系统本身的防护还远远不够。还需要搭配许多常用的软件和工具，才能尽量保证各个方面的安全。
