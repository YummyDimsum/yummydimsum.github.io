    使用 VPN 的场景 #

    一、为什么使用 VPN #

    对于许多中国用户来说，提起使用 VPN, 首先想到的就是“翻墙”。

    的确，VPN 可以帮忙翻墙，但这并不是它唯一的用途，我们在其它很多场景之下都需要使用 VPN.

    1. 在公共的网络环境里 #

    比如外出住酒店时，使用酒店的 WiFi， 这时就应该全程开着 VPN, 让自己的网络流量加密传输，以免被同一个局域网里的人（也就是其他连接到酒店 WiFi 的人）窥探。在咖啡馆、机场等处，也是同样的道理。

    2. 隐藏真实的 IP 地址 #

    比如你常住在美国，从事一些线上匿名活动，有一些匿名的网络账号，你不希望服务商（比如 Gmail）知道你在美国。这时你就可以选择美国以外的 VPN, 比如欧洲或者东南亚，假装自己来自这些地方。

    3. 帮助你突破网络封锁 #

    这也就是所谓的翻墙。这一点大家都已经很熟悉，这里就不过多展开。

    由此可见，无论你是否生活在中国境内，都建议你使用 VPN 上网，它能更好地保护你的安全与隐私。

    下面先一些集成式的 VPN 软件和服务（也就是下载、安装之后能够直接用的），再介绍怎样自己搭建 VPN 服务。

    二、在中国使用 VPN #

    由于中国严酷的网络封锁，选择合适的 VPN 变得异常困难。很难说有哪一种 VPN 在中国能够长期稳定且快速地使用。在中国使用 VPN 需要技术，更需要耐心。

    迷雾通 Geph #

    https://geph.io/zhs

    开放源代码，公司注册在爱沙尼亚，属于欧盟公司。欧盟有全世界最严格的数据隐私保护法。

    迷雾通支持所有操作系统，除了 iOS. 免费版的速度相对较慢，收费版每月 5 欧元，它有一种特殊的机制，保证用户付款过程中不会泄漏真实身份。

    蓝灯 Lantern #

    https://lantern.io/zh-Hans

    支持所有操作系统。尽管在中国一些地区连接困难，但在有些地区仍然可以使用。

    Orbot #

    https://orbot.app/en/

    Guardian project 所开发的、基于 Tor 网络的手机 VPN 软件，支持 iOS 和 Android 系统，免费使用。

    因为基于 Tor 网络，所以它可以将网络请求和流量经过 Tor 网络转发，在帮助用户突破网络封锁的同时也可以保持匿名。同时它还能够只针对特定的 App, 而不是针对整个系统代理。在中国使用的话，需要配合内置的 Snowflake 功能连接，或者手动配置网桥。关于 Tor 网络的介绍，请看下一小节《使用 Tor 浏览器保持匿名》。

    三、在中国之外使用 VPN #

    能够在中国使用的 VPN，也都能在中国之外使用，除此之外，还有一些在海外比较流行的 VPN。

    Proton VPN #

    https://protonvpn.com/

    Proton mail 同一家的服务。支持所有操作系统。它的免费版本提供有限的服务器，收费版则可以使用全球各地的服务器。

    Riseup VPN #

    https://riseup.net/en/vpn

    支持 Android, Windows, macOS 和 Linux 系统，不支持 iOS 系统。

    Express VPN #

    https://www.expressvpn.com/

    支持所有操作系统，只有收费版本，100 美元一年。

    Mullvad VPN #

    https://mullvad.net/

    支持所有操作系统，只有收费版本，60 欧元一年。

    四、自己搭建 VPN 服务 #

    【提醒】在中国搭建 VPN 比单纯使用 VPN 更有风险，尤其是搭建 VPN 之后再与其他人分享，可能会导致警察上门甚至抓捕判刑。

    Shadowsocks #

    https://gfw.report/blog/ss_tutorial/zh/

    Shadowsocks 曾经是非常流行的 VPN 软件，后来由于 GFW 不断探测和封锁，使用人数逐渐变少。但它的开发者社区一直在与 GFW 斗智斗勇，因此目前仍然有办法规避和使用。

    以上链接不是 Shadowsocks 的项目页面，而是 GFW Report 项目撰写的一份报告：《如何部署一台抗封锁的 Shadowsocks-libev 服务器》。

    有各种不同的软件和应用都支持 Shadowsocks, 在你决定使用某个软件之前，请确认它：是开源软件，而且过去没有不良的记录。

    Outline VPN #

    https://getoutline.org/

    这由 Google 资助、基于 Shadowsocks 协议开发的开源 VPN 软件。它本身只是一个软件，没有 VPN 账号。用户可以自己搭建 Outline 的服务器来创建账号，也可以从其他地方获取或购买账号。

    如果想要自己搭建 Outline VPN 服务，可以参考这个教程：

    https://matters.town/@fastlane982/36407-%E4%BD%BF%E7%94%A8-outline-%E5%9C%A8%E4%B8%AD%E5%9B%BD%E6%90%AD%E5%BB%BA%E5%AE%89%E5%85%A8%E9%AB%98%E9%80%9F%E7%9A%84-vpn-%E7%9A%84%E7%AE%80%E8%A6%81%E6%8C%87%E5%8D%97-bafyreic4y5go4bi5d4ozrplqxs2dye336odhxoo6rjzgrsqkrj5hkipv44

    V2Ray #

    https://www.v2ray.com/index.html

    这是一个突破网络审查和封锁的平台。据测试发现，使用 V2Ray 加上 WebSocket + TLS 混淆流量，可以相对比较稳定地翻墙。

    在不同操作系统上有不同的 V2Ray 客户端。在你决定使用某个软件之前，请确认它：是开源软件，而且过去没有不良的记录。用户需要自己搭建服务器来创建账号，也可以从其他地方获取或购买账号。

    【请注意】网上有很多“机场”，出售 Shadowsocks / Outline / V2Ray 账号。我们在这里郑重提醒：请一定不要从不明来源的人和网站购买账号，因为你的上网记录可能会被监控和记录，从而给你带来极大风险。

    五、谨防钓鱼 VPN #

    过往案例表明，有关部门会故意搭建一些“钓鱼 VPN”，诱骗用户上当。

    我们无法罗列所有的钓鱼 VPN 名称，也没法告诉你如何准确判断哪些是钓鱼 VPN。

    我们能给的建议是，尽可能使用开放源代码的 VPN 软件，不要从陌生人手中购买账号，尤其不要贪图便宜，使用网上到处流传的、不知来源的免费账号。
