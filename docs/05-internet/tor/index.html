    使用 Tor 浏览器保持匿名 #

    当我们说“使用 Tor 浏览器保持在线匿名”的时候，先要区分清楚，你什么时候想保持匿名，什么时候想保证数据加密，这两者不一样。

    当你想在网络上匿名的时候，主要是不想让别人（无论是警察还是网站的管理员）知道是你在使用某个网站或服务。匿名上网时，除了不能透露任何真实的个人信息，最好也不要透露你的电子设备（包括操作系统、浏览器版本、网络连接状态等）信息。

    Tor 浏览器是基于 Firefox 浏览器而开发的、专门保护用户匿名的浏览器。

    一、工作原理 #

    Tor 网络在全世界范围内有成千上万的节点，用于转发用户的网络请求和流量。当用户使用 Tor 浏览器上网的时候，它会自动为用户选择三个服务器转发流量。

    比如，当你访问 Facebook 的时候，Tor 浏览器可能会帮你连接到加拿大→德国→美国的节点，最终，Facebook 看到的是一位来自美国的用户正在访问。

    当你在同一个浏览器里访问另一个网站的时候（比如 Google），Tor 浏览器又会为这个目标网站选择新的三个转发节点，可能会是加拿大→卢森堡→荷兰，于是 Google 看到的是一位来自荷兰的用户正在访问。

    使用 Tor 浏览器要特别留意的是，任何涉及个人真实身份信息和隐私信息的内容，不要直接通过 Tor 浏览器转发，比如不要使用它登录 Amazon 购物，更不要在使用 Tor 浏览器的时候输入信用卡信息，因为这些都会关联到你的真实身份。

    三、中文指南 #

    由于 Tor 浏览器的官方网站在中国被封锁，它的所有公开节点（relay）也在中国被封锁，中国用户想要使用 Tor 浏览器和 Tor 网络就比较困难。

    Tor 官方提供一份针对中国用户的指南，可以帮助你下载安装 Tor 浏览器，并且成功连接到 Tor 网络。 https://github.com/torproject/tor4zh

    还有一个中文的长毛象账号，会经常发布 Tor 浏览器的使用方法。 https://mastodon.social/@tor4zh
