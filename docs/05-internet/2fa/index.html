    为你的账号开启两步验证 #

    一、什么是两步验证 #

    中国的互联网服务普遍没有登录的两步验证（2FA, Two-Factor Authentication）功能，但主流的国际互联网服务都有这一选项。

    所谓两步验证，就是在登录的时候，输入用户名和密码之后，还需要输入一串验证码。

    两步验证分为几种方式：

    二、使用手机号码接收验证码 #

    这种方式并非最可靠的，但有些网站只提供这种验证方式。

    这里再次提醒，因为你注册的是不同的在线虚拟身份，所以一定不要使用已经实名的中国手机号码。如果你使用国外的手机号码，使用手机号码接收验证码并无不妥。

    三、使用专门的两步验证 App #

    有些网站提供一种功能，允许你在手机上安装 App, 绑定这个网站，每次登录的时候，两步验证 App 上面会自动生成六位数字，有效期仅有一分钟。

    • Android 用户，推荐使用 Aegis Authenticator；

    • iOS 用户，推荐使用 Tofu.

    如果你不介绍使用 Google 家的服务，那么在 Android 和 iOS 上面都有 Google Authentication 可用。

    • Android https://play.google.com/store/apps/details?id=com.google.android.apps.authenticator2

    • iOS https://apps.apple.com/us/app/google-authenticator/id388497605
