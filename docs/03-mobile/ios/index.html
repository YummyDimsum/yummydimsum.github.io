    iPhone 手机 #

    与其他国家（比如美国）销售的 iPhone 相比，中国国行的 iPhone 手机有一些功能受限。这些限制，有些属于硬件层面，有些属于系统层面，有些属于内容层面。

    • 硬件层面：中国国行 iPhone 没有 eSIM 功能，也就是它只能使用实体 SIM 卡。

    • 系统层面：中国国行 iPhone 不能使用 FaceTime 音频通话而只能视频通话，没有 CallKit, Walkie-Talkie, 5G 毫米波、卫星紧急救援等功能，同时 Apple Pay 功能受限。

    • 内容层面：书籍、音乐、电影、应用程序等许多方面遭到审查，详见下文。

    作为普通用户，你很难自行解决硬件层面和系统层面的限制问题，但可以一定程度解决内容限制问题。

    一、中国区 Apple ID 的限制 #

    iPhone 用户下载安装 App 的正规渠道（也是相对安全的渠道）就是苹果公司自家运营的 App Store。想要使用 App Store, 需要注册一个 Apple ID。事实上，不光是 App Store，iPhone 上面的许多 Apple 服务都需要 Apple ID 登录。

    如果你直接在中国境内注册 Apple ID（即没有使用任何其它辅助措施），那么你的 Apple ID 会被归类为“中国区 Apple ID”。中国区 ID 与其他 ID 有三大不同：

    1. 使用中国区 Apple ID 登录 App Store, 只能搜索到有限的应用。据媒体报道，在中国区的 App Store，上万个应用遭到下架，包括外国新闻机构、翻墙、政治、宗教等，甚至包含教育、游戏类应用。使用中国区 Apple ID 在 App Store 搜索，找不到这些应用。从某种程度上说，这意味着你的 iPhone 手机无法连接到国际互联网。

    2. 中国区 Apple ID 在内容方面也遇到严重的审查和限制。Apple Books, Apple Music, Apple News, Apple Podcasts, Apple TV, Apple TV+, iTunes 等服务都有不同程度受限。

    3. 如果你使用苹果公司的 iCloud 云端存储服务，中国区 Apple ID 用户的 iCloud 数据会被存储在中国贵州省的数据中心。这意味着中国警方在他们需要的时候可以查看你的数据，尽管苹果公司声称数据已经加密。《纽约时报》对此有一篇详细的报道。

    以上这些问题给中国区 Apple ID 用户造成很多不便甚至风险，尤其当你是记者、律师、公民行动者或人权捍卫者的时候。如果无法正常使用国际互联网的服务，或者你把数据存储于中国的数据中心，都有可能给你带来人身危险。

    我们的建议是：要么不使用 iPhone 手机，要么为你的 iPhone 注册其他地区的 Apple ID（比如美国地区的 Apple ID），不要将数据存储在中国的 iCloud 数据中心。

    在中国注册美国区 Apple ID 的方法很多，比如这里的指南：

    https://sspai.com/post/83617

    【请注意】在中国注册美国区 Apple ID 的方法并不能保证总是有效。建议你在操作之前搜索寻找最新的方法，并且完整阅读操作步骤，确保完全理解其中的每一个步骤，再去操作。注册美国区 Apple ID 即使失败也不会损害你的手机，所以请大胆尝试。

    二、AirDrop 隔空投送信息的风险 #

    特別提醒 AirDrop 传送文件的安全性问题。

    2024 年初，北京市司法局称，与其合作的司法鉴定机构已经能够追踪 AirDrop 隔空投送信息的发送者手机号和邮箱账号。这基本上意味着能够追踪到发送者的真实身份。

    iPhone 的 AirDrop 能在苹果设备之间匿名共享文件，曾经有人利用苹果设备的这一功能在公共场所向陌生人传播社会运动、公民运动相关信息。

    详情请见澎湃新闻相关报道： https://www.thepaper.cn/newsDetail_forward_25935720
