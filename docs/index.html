    写在前面 #

    尽管已经有许多手册和指南帮助大家提升信息安全和网络安全，但这些指南大都用英文写成。中国的政治环境、言论环境和互联网现状不同于其他国家，即使将那些英文指南全部翻译成中文，可能也无法完全适合中国读者。因此我们决定撰写一个专门针对中国读者的安全手册。

    本手册的初衷是帮助中国的记者、律师、公民行动者和广义的人权捍卫者等“高危人群”，提升他们的信息安全和网络安全，因此我们重视“安全”和“隐私”多于“方便”和“简单”。但我们也知道，过于复杂的方案和工具会阻碍大家的使用热情，因此我们会努力平衡“安全”与“方便”两方面。

    撰写一份适合所有人的指南是不可能的，我们只能努力使其难易适中。如果读者认为哪些地方过于简略、不知如何操作，或者哪些地方难以理解，欢迎给我们多提意见。你可以在这些地方找到我们。
